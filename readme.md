### อุปกรณ์ที่ใช้
* ESP32-S3 Devkit 2 ตัว
(ตัวที่ 1 สำหรับเชื่อมกับจอ
,ตัวที่ 2 สำหรับเชื่อมกับ RFID READER เพื่อสแกนบัตร)
* RFID READER 1 ตัว เพื่ออ่านบัตร
